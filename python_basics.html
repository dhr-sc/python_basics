In this exercise, you will use Python to download data and metadata from ERDDAP.

The exercise demonstrates the following skills:

  • Using Python to retrieve information about a dataset from ERDDAP
  • Getting data from a remote CSV file
  • Downloading satellite data from ERDDAP in netCDF format
  • Extracting data with Python

For the examples in this exercise we will use the SMAP Sea Surface Salinity dataset from the CoastWatch West Coast Node (ERDDAP ID = jplSMAPSSSv42_mday).
https://coastwatch.pfeg.noaa.gov/erddap/griddap/jplSMAPSSSv42_mday.graph

The dataset has low spatial resolution (0.25 degrees), which will help prevent overloading the local intenet bandwidth during the class.

The scripts in this exercise are written in Python 3.7.

Getting information about a dataset¶

An ERDDAP has an interface for gathering information about any or all datasets¶

  • Go to the ERDDAP Active Datasets page
    https://coastwatch.pfeg.noaa.gov/erddap/tabledap/allDatasets.html
    The page shows you all of the metadata that you can collect for any or all datasets.
  • Use the Uncheck all button to deselect all of the metadata variable
  • Check the datasetID row. In the Optional Constraint #1 column change the dropdown to = and paste in the dataset ID with double quoates around it: "jplSMAPSSSv42_mday"
  • Check the following rows: 'minLongitude', 'maxLongitude', 'minLatitude', 'maxLatitude', 'minTime', and 'maxTime'
  • Now click the Submit button

ERDDAP loads a page showing information about the dataset. The information is in HTML format. We want to get it in CSV format to load it into Python.

  • Use the browser back button to return to the ERDDAP Active Datasets page.
  • Use the file type dropdown (just above the Submit button) to select a csv file type.
  • Then click the Just generate the url button and copy the URL.

In the script below the URL we generated in ERDDAP is pasted into the myurl variable. The script uses this URL to bring the metadata into a Pandas dataframe.

Load data for all variables into Python arrays¶

To use the data form the netCDF file, we need to load it into variables.

  • Use the nc.variables method.

  • You need to specify what data you want from each variable array.

    • The smap_sss variable is 3D, so by specifying nc.variables['smap_sss'][:,:,:] you get all of the data. If you used nc.variables['smap_sss'][0,:,:] you would only get the first time step. If you used nc.variables['smap_sss'][1:2,:,:] you would only get the second and third time step.

    • The coordinate variables (time, latitude, longitude) are 1D, so nc.variables['latitude'][:] loads all of the data.

In [10]:
sss_data = nc.variables['smap_sss'][:,:,:]
lat_data = nc.variables['latitude'][:]
lon_data = nc.variables['longitude'][:]
time_data = nc.variables['time'][:]

# Close the netCDF file
nc.close()

You are done! You have successfully loaded ERDDAP satellite data into Python.¶

Create a function that downloads data and generates means¶

The function incorporates the URL generation, ERDDAP data requests and mean calculations we used above.

In [24]:
def get_open_data(start_date, end_date, lat_min, lat_max, lon_min, lon_max):
    # create the URL
    base_url = 'https://coastwatch.pfeg.noaa.gov/erddap/griddap/'
    id_url = 'jplSMAPSSSv42_mday'
    url_filetype = '.nc?'
    var_url = 'smap_sss'
    time_url = '[(' + start_date + '):1:(' + end_date + ')]'
    lat_url = '[(' + lat_max + '):(' + lat_min + ')]'
    lon_url = '[(' + lon_min + '):(' + lon_max + ')]'
    
    the_url = ''.join( [base_url, id_url, url_filetype, var_url, time_url, lat_url, lon_url] )
    print(the_url)
    
    # Download the data
    try:
        print('loading')
        urllib.request.urlretrieve(the_url, 'myNC2.nc')
        print('data')
        nc2 = netCDF4.Dataset('myNC2.nc', "r")
    except:
        print('No ERDDAP responce')
    
    # bring data into python
    sss_dt = nc2.variables[var_url][:,:,:]
    lat_dt = nc2.variables['latitude'][:]
    lon_dt = nc2.variables['longitude'][:]
    #time_dt = nc2.variables['time'][:]
    
    nc2.close()
    
    # Generate means
    m_mean = np.nanmean(sss_dt, axis=(1,2))
    m_mean_print = list(np.around(m_mean, decimals=3, out=None))

    m_std = np.nanstd(sss_dt, axis=(1,2))
    m_std_print = np.around(m_std, decimals=3, out=None)
    m_std_print = [str(ln) for ln in m_std_print]

    m_cnt = np.count_nonzero(sss_dt, axis=(1,2))
    m_cnt_print = [str(ln) for ln in list(m_cnt)]
    
    return lat_dt.min(), lat_dt.max(), lon_dt.min(), lon_dt.max(), m_mean, m_std, m_cnt
